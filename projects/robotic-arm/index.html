/ OVERVIEW

INTEGRATED
SIMULATION

This project implements a complete simulation environment for a 6 Degree-of-Freedom serial robotic arm using ROS (Robot Operating System) and Gazebo. The arm is modelled using URDF with accurate joint limits, inertial parameters, and collision geometry derived from a real hardware design.

MoveIt serves as the motion planning framework — enabling inverse kinematics solving, collision-aware trajectory generation, and execution via the ROS control interface. The simulation validates path planning algorithms before any hardware deployment, dramatically reducing iteration time.
